Erratum number 8
Entry date 2009-12-14
Spec document IPTC Photo Metadata (July 2009)
Issue

IPTC Extension property: Digital Source Type
Current Help Text: Select one of the values for identifying the source of the digital image from the controlled vocabulary.
Issue: not the source but the type of source should be identified!
Planned correction Change the help text to:
"Select one of the values for identifying the type of the source of the digital image from the controlled vocabulary"
Note corrected in the spec document version (July 2010)
